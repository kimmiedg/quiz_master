Welcome to Quiz Master V.1! 
Before running the application, please be reminded by the following: 

* Ruby version, ruby 2.2.3

* Rails version, Rails 4.2.4

* Database initialization
	- Kindly do, rake db:create db:migrate db:seed

* Run/Test
	- In order to create categories and or questions, the user must be an administrator. Kindly see the seeds.rb file to see the admin credentials.
